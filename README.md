# go / gset

To install `gset/go` function in a zsh command_prompt
 
```
$ git clone git@github.com:QDucasse/zsh.git
$ cd zsh
$ cat ~/.bashrc go_gset > ~/.bashrc
$ cat ~/.zshrc  go_gset > ~/.zshrc
```

Usage:

```
$ cd pathToTheBookmark
$ gset bookmarkName
```
then from anywhere on your computer
```
$ go bookmarkName
```
